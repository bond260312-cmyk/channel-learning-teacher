桌面同步需求清單

這頁不是工程實作，而是把桌面版要同步什麼先講清楚，避免後面越做越亂。

回課程入口 回課程總綱

第一版一定要同步的 6 件事

  1. 目前看到哪一課： last opened lesson。
  2. 每課完成狀態： 未開始 / 進行中 / 已完成。
  3. Lesson 1 的自述內容： 你先說與複習卡內容。
  4. 測驗結果： 最近一次得分、是否過門檻。
  5. 課程版本： 避免舊快取蓋掉新內容。
  6. 本機時間戳： 判斷最後更新來源。

建議同步欄位

courseId
lessonId
status
lastOpenedAt
quizScore
quizPassed
selfExplain
reviewCard
contentVersion
updatedAt
syncSource (desktop / local)

同步策略建議

  • 第一版先用 localStorage / JSON 檔，別一開始就上後端大工程。
  • 寫入時採 整課覆蓋，不要拆太細，減少同步錯誤。
  • 若版本衝突，先顯示「本機較新 / 遠端較新」，不要默默覆蓋。
  • 先完成單機桌面同步，再談多裝置帳號同步。
